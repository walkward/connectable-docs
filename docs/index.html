    Connectable API

    Base URLs:

    • https://connectable.heroku.com/api

    License: Apache 2.0

    Authentication

    • oAuth2 authentication. HTTP Bearer authentication strategy using Passport.

      • Flow: implicit
      • Authorization URL = https://connectable.auth0.com
    Scope Scope Description
    read:users Get a users information.
